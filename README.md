## Featured Projects

### [Custom Emoji Studio](https://github.com/210on/custom-emoji-studio)

A web app for designing beautiful custom emoji with immediate visual feedback.  
Built as a research-oriented frontend prototype around readability and small-size visibility.

### [OpenSCAD Customizable Coffee Dripper](https://github.com/210on/openscad-customizable-coffee-dripper)

Parametric OpenSCAD experiments for coffee dripper design.  
Focused on geometry control, prototyping, and practical fabrication.

### [OpenSCAD Hybrid Brewer](https://github.com/210on/openscad-hybrid-brewer)

An experimental brewing-device modeling project in OpenSCAD.  
Used to compare structure, cross-sections, and manufacturable forms.

### [Paragon Stand Extention Stem](https://github.com/210on/paragon-stand-extention-stem)

A parametric extension stem model for a Paragon stand.  
Organized as an adjustable OpenSCAD model for iterative physical testing.
